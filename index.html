Elden ring

Le château de Valorage est une zone importante dans Elden Ring, situé après la région de Nécrolimbe. Après avoir vaincu Margit le déchu, le joueur peut entrer dans le château de Valorage où il rencontrera le Chevalier du Creuset, un boss qui peut être trop difficile pour le joueur à ce stade du jeu. Dans le château de Valorage, le joueur peut rencontrer Roderika dans la cabane de Valorage, qui peut lui donner l'emote "S'asseoir de côté" et les Cendres de Méduse Spirituelle. Le joueur peut également récupérer la Clé-Lame de Pierre sur un cadavre dans la zone. Après avoir exploré la zone, le joueur peut se rendre à la cour du château où il devra faire face à un groupe d'ennemis postés en haut de la pente. Le joueur peut invoquer des nobles pour aider à vaincre ces ennemis.
